# Project Requirements - Nexa.luxury

## Goal
To provide an educational, premium, and performant web platform explaining the Nexa cryptocurrency, its supply mechanics, and how it compares to Bitcoin.

## Core Features
1.  **Educational Content**: Clear explanations of Nexa, its supply (21 Trillion), divisibility, and halving schedule.
2.  **Interactive Elements**: Smooth animations (starfield background, hover effects).
3.  **Modern Tech Stack**: Static site built with SCSS, ESBuild, and optimized for speed.
4.  **Advanced SEO**: 
    - Full Meta tags (Title, Description, Keywords).
    - OpenGraph and Twitter cards.
    - Sitemap and Robots.txt.
    - AI Search Engine Optimization (JSON-LD, llms.txt).
5.  **Premium Aesthetics**: Dark mode by default, vibrant accents (#efc426), and high-quality typography (Inter, Space Grotesk).
6.  **Mobile-First Design**: Fully responsive layout optimized for all devices.

## Performance Requirements
- Optimized assets (minified CSS/JS).
- Fast loading times (low HTTP requests).
- Accessibility (Aria labels, semantic HTML).
